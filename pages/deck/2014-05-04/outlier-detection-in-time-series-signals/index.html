Outlier Definition

An observation in a data set which appears to be inconsistent with the remainder of that set of data

  • Easy to handle if it is generated by a regular process

  • ## Median Filter $$ y[m] = median\\{ x[m-\frac{w}{2}, m+\frac{w}{2}] \\} $$ where $w$ is the window size whose median will replace the original data point. #### Properties - Not linear - Does not smooth the signal. (Does not blur the images) - Robust to outliers!
